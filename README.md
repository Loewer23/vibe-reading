# Vibe Reading 2.0 · 氛围阅读

> **AI 辅助深度阅读技能**  
> 专为英文原版书、社会科学经典（历史 / 政治 / 社会学 / 人类学 / 公共管理）而设计。  
> 让你从「翻开就困」到「读透、批判、输出」。

## 🎯 解决什么问题

- ❌ 读不懂外文书、社科专著  
- ❌ 读完了像没读，记不住核心论证  
- ❌ 不知道如何批判性思考  
- ❌ 术语太多，学术语境陌生  
- ❌ 读完无法输出、不能用于论文或工作

## ✨ 核心特性

| 模块 | 作用 |
|------|------|
| **量化认知负荷** | 轻松 / 进阶 / 深度三档，单次输出 ≤300~1000 字，不 overload |
| **闭环对话教学** | 每段讲解后强制开放式提问，拒绝 AI 腔 |
| **纵向阅读法** | 拆解结构脉络，还原论证链条 |
| **横向阅读法** | 跨文本对比学派、跨学科映射现实 |
| **词源学还原** | 外文关键词追根溯源，避免翻译失真 |
| **知识债务清算** | 识别作者的「默认假设」，破坏测试逻辑漏洞 |
| **自动化成果输出** | 概念卡 / 思维导图 / 对比表 / 批判清单 |
| **学术严谨管控** | 明确区分作者观点、AI 解读与第三方观点 |

## 🚀 快速开始（30 秒上手）

### 1️⃣ 将以下 Prompt 发给任意 AI 助手

> 推荐使用 **ChatGPT (GPT-4/4o)、Claude 3.5+、DeepSeek** 等支持长上下文的模型。

将本仓库中的 **`vibe-reading-prompt.md`** 全文作为「系统指令」或第一次消息发送。

### 2️⃣ 使用触发词唤醒技能

发送下列任意一条消息：

- `vibe reading` / `氛围阅读`  
- `教我读书` / `帮我读书` / `读书指导`  
- `外文书阅读` / `社科书籍阅读`

### 3️⃣ 回答 AI 的 3 个启动问题

AI 会自动询问：

1. 阅读内容类型（外文书 / 教材 / 论文 / 节选）  
2. 阅读目标（入门 / 深度吃透 / 备考 / 批判）  
3. 难度档位（轻松 / 进阶 / 深度）

回答后，你的专属阅读教练便启动。

## 📖 使用示例

**用户**  
`vibe reading`  
《The Interpretation of Cultures》格尔茨，想深度吃透，进阶档。

**AI（按技能执行）**  
> 好，咱们今天啃这本厚书。先给你一张地图：这本书的核心就一个问题——文化到底该怎么「被研究」？  
> 格尔茨反对把文化当「行为规律」或「心理结构」，他提出了 **thick description**（深描）……  
> [输出控制600字以内，补充词源学：description 不是简单描述，而是「层层剥开意义」]  
> 来，第一个互动：你第一次听到「深描」这个词，脑海里冒出的场景是什么？  
> —— 随便说，咱们用它来怼书里的例子。

**用户**  
（回答后）AI 继续推进 → 清算默认假设 → 生成概念卡 → 横向对比 vs 列维-斯特劳斯 → 最终产出全书脉络图。

## 🗂️ 适合的书籍类型

- 英文原版社科 / 人文著作  
- 社会学、人类学、政治学、历史学经典  
- 高校教材（如《政治学的邀请》《社会学的想象力》）  
- 学术论文（单篇或论文集）  
- 任何让你「读不下去」的外文或社科文本
